BBox.of unions every polygon of a state into one box, so a state whose polygons are geographically dispersed (Hawaii is many islands; Alaska plus outlying isles) gets a loose box that many non-contained points still pass, forcing the full ray-cast. A per-polygon bbox prefilter (reject each polygon by its own box before ray-casting it) would reject more points cheaply. Minor for 51 US states, but it is the selectivity knob if the geometry set grows.
